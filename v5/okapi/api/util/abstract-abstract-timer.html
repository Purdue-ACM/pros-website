okapi::AbstractTimer¶

A time- and timing-related utility class. Implements its methods in terms of millis().

Prototype

repeat¶

Returns true when the input time period has passed, then resets. Meant to be used in loops to run an action every time period without blocking.

Prototype
Example
virtual bool repeat(QTime time)

void opcontrol() {
  okapi::Timer timer;
  while (true) {
    if (timer.repeat(100_ms)) {
      // Do something every 100 ms
    }
    pros::delay(10);
  }
}
Parameters  
time The time period.

Returns: true when the input time period has passed, false after reading true until the period has passed again

repeat¶

Returns true when the input time period has passed, then resets. Meant to be used in loops to run an action every time period without blocking.

Prototype
Example
virtual bool repeat(QFrequency time)

void opcontrol() {
  okapi::Timer timer;
  while (true) {
    if (timer.repeat(10_Hz)) {
      // Do something 10 times per second
    }
    pros::delay(10);
  }
}
Parameters  
frequency The repeat frequency.

Returns: true when the input time period has passed, false after reading true until the period has passed again

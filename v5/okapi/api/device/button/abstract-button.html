isPressed¶

Returns whether the button is currently pressed.

Prototype

Returns: Whether the button is currently pressed.

Returns whether the state of the button changed since the last time this method was called.

Prototype
virtual bool changed()

Returns: Whether the state of the button changed since the last time this method was called.

changedToPressed¶

Returns whether the state of the button changed to pressed since the last time this method was called.

Prototype
virtual bool changedToPressed()

Returns: Whether the state of the button changed to pressed since the last time this method was called.

changedToReleased¶

Returns whether the state of the button changed to not pressed since the last time this method was called.

Prototype
virtual bool changedToReleased()

Returns: Whether the state of the button changed to not pressed since the last time this method was called.

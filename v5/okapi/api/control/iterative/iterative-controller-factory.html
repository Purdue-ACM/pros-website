okapi::IterativeControllerFactory¶

A factory which creates instances of iterative controllers. This is the preferred way of creating iterative controllers.

You can read more about the factory pattern here.

Methods¶

Prototype

Parameters  
ikp The P term gain.

ikD The D term gain.

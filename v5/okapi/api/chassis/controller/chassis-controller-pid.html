okapi::ChassisControllerPIDÂ¶

A ChassisController using PID control. Does not use the V5 motorâs integrated control. The motors passed in will be put into degree units. If you are trying to make an instance of this class, you should most likely be using the ChassisControllerFactory instead of a constructor from this class. Throws a std::invalid_argument exception if the gear ratio is zero.
